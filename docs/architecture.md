# NeuroMark 首版架构

## 技术栈
- Electron 负责桌面壳、文件系统、SQLite、窗口与后台任务。
- Vue 3 + Vite + TypeScript 负责渲染层。
- Naive UI 负责桌面中文界面。
- Better SQLite3 + Drizzle ORM 负责全局索引数据库。

## 数据分层
- 项目数据以项目目录为中心，保存原图、扫描图、评分结果与导出文件。
- 全局数据库位于 Electron `userData`，保存项目索引、后台任务、全局设置与参考答案生成历史。

## 运行时边界
- 主进程负责所有有权限和副作用的能力。
- 渲染进程只通过 `window.neuromark` 调用受控 API。
- 预览窗口复用同一套渲染代码，通过单独路由展示大图浏览器。

## 当前阶段
- 首版主要完成结构、布局、Mock 数据链路与可扩展 IPC 骨架。
- 扫描与批阅任务先以模拟任务形式展示状态流转，为后续真实接入 OpenCV 与多模态 LLM 预留接口。
